deep-research

Fan-out web research workflow that cross-checks cited claims by vote and returns a sourced report.

When to use

Use for research questions that need multiple independent angles and a verification pass — comparisons, “state of X”, due-diligence. It plans angles, searches each in parallel, extracts cited claims, cross-checks them by vote, and synthesizes a report with sources. Not for quick single-fact lookups.
